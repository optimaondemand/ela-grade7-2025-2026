# 7th Grade ELA: The Art of Story & Argument
**Course:** English 7 (M/J Language Arts 2) | **Grade:** 7th | **Duration:** Two Semesters (~36 weeks) | **Trivium Stage:** Logic Stage

---

## Course Overview

This two-semester asynchronous course builds on the narrative and grammar foundations of 6th grade and prepares students for the rhetorical demands of 9th. Students read enduring literary works — Homer, Tolkien, Taylor, Shakespeare, Orwell, Frank — and develop the analytical vocabulary to argue about them in writing. The central thread from semester to semester: *What is a story for, and what does it ask of its reader?*

**Year Essential Questions:**
- What makes a person heroic — and what unmakes them?
- How do stories shape what we believe about the world?
- Can words change minds, or do they only reflect what minds already hold?

**Trivium stage:** Logic (grades 6–8). Students move from narration toward argument, from "what happened" toward "what it means and why it matters." Every module develops both close reading and structured written reasoning. Grammar is taught as a tool for sharpening thought, not as a separate drill.

## Grading Architecture — Built for 500 Students

This course is designed for an asynchronous facilitator carrying approximately 500 students. Every graded item fits one of three tiers. **The ratio of teacher time to graded work is deliberately protected.**

| Tier | Type | Target % | Teacher Hours |
|------|------|----------|---------------|
| **Tier 1** | Auto-graded (Canvas Quiz, completion-checked assignment) | ~65% | 0 |
| **Tier 2** | Structured discussion + module outro quiz triage | ~20% | 2–3 hrs/week |
| **Tier 3** | ONE module project per module with rubric | ~15% | 8–12 hrs/module cycle |

### Per-Module Point Breakdown (5-week module)

| Item | Tier | Pts | Notes |
|------|------|-----|-------|
| Weekly Reading Quiz × 5 | T1 | 50 | Auto-graded, 10 pts each |
| VR Activity 1 × 5 | T1 | 50 | Completion-graded, 10 pts each |
| VR Activity 2 × 5 | T1 | 50 | Completion-graded, 10 pts each |
| Quick Write × 3 (wks 1–3) | T1 | 15 | Completion only (5 pts each) |
| Module Discussion (mid, wk 3) | T2 | 15 | Rubric: substantive=15, thin=8, missing=0 |
| Module Discussion (close, wk 5) | T2 | 15 | Same rubric |
| Module Project | T3 | 50 | Canvas rubric, 5 min/student in SpeedGrader |
| Module Intro Quiz | T4 | 0 | Ungraded — analytics only |
| Module Outro Quiz | T4 | 5 | Completion credit — teacher reads trends |

**Module total: 250 pts**
- Tier 1: 165 pts = 66% ✓
- Tier 2: 30 pts = 12%
- Tier 3: 50 pts = 20%
- Tier 4 (ungraded scaffolding): 5 pts completion

> **Teacher time per module cycle (~5 weeks):**
> - Tier 1: 0 hours
> - Tier 2 discussions: ~2 hrs/week (500 × 30s per post, read in batches)
> - Outro quiz triage: 20 min (scan trends, drop 2–3 public clarifications)
> - Tier 3 project: ~42 hours total spread over 2 weeks (500 × 5 min per SpeedGrader submission)

## Semester 1: The Art of Story

*Essential Question: What is a hero, and what does it cost to become one?*

### Module 1 — The Odyssey (5 weeks)
**Text:** Homer, *The Odyssey* (Fitzgerald translation, selected books: 1, 5, 9, 10, 11, 12, 21–22)
**Central Question:** What does it mean to be cunning? Is cleverness a virtue or a vice?
**Themes:** Hubris and its cost, loyalty and betrayal, the hero's long road home
**Grammar Focus:** Gerund phrases, appositive chains, subordinate clause types
**VR Arc:** ENGAGE Ancient Greece/Aegean settings — investigation, scene composition, project assembly
**Module Project:** Epic hero analysis essay + ENGAGE spatial composition (Odysseus's defining trial)

| Week | Chapters/Books | Main Lesson | Grammar 1 | Grammar 2 | VR 1 | VR 2 | Quick Write/Discussion |
|------|---------------|-------------|-----------|-----------|------|------|------------------------|
| 1 | Bks 1, 5 | The Epic Hero & His World | Gerund phrases | Infinitive phrases | Aegean Setting Investigation | Hero Composition Draft | QW: What kind of man is Odysseus? |
| 2 | Bks 9–10 | Cunning, Hubris, & the Cyclops | Appositive chains | Relative clauses (who/which/that) | Cyclops Cave Staging | Cunning vs. Strength Composition | QW: When does cleverness become cruelty? |
| 3 | Bks 11–12 | The Underworld & The Cost of the Journey | Subordinate clauses (review) | Adverbial clauses of concession | Underworld Annotation Walk | Cost of Hubris Composition | **Discussion (mid-module)** |
| 4 | Bks 21–22 | Homecoming — Justice or Revenge? | Parallel structure (review) | Sentence-level rhetoric: anaphora | Homecoming Scene Tableau | Evidence Composition | QW: Is Odysseus's revenge just? |
| 5 | Bks 21–22, full review | The Epic in Retrospect | Periodic sentences | Cumulative sentences | Project Assembly Session | Peer Review Gallery | **Discussion (module close)** |

## Canvas Implementation Guide

### Step 1 — Upload PDFs and Texts to Canvas Files
1. In Canvas, go to **Files** (left sidebar).
2. Create a folder called `ELA Resources — Grade 7`.
3. Upload the following (PDFs should be licensed copies):
   - Odyssey selections (Fitzgerald translation)
   - *The Hobbit* PDF
   - *Roll of Thunder, Hear My Cry* PDF
   - Poetry anthology selections
   - *The Diary of a Young Girl* PDF
   - *Animal Farm* PDF
   - *Romeo and Juliet* PDF (Folger or equivalent)
   - Rhetoric unit texts (all public domain — use direct links)
4. Copy the Canvas file link for each text after upload.

### Step 2 — Create Canvas Modules (one per module)
1. Go to **Modules** → **+ Module**.
2. Name each module: `Module 1: The Odyssey`, `Module 2: The Hobbit`, etc.
3. Set prerequisites if desired (Module N must be completed before Module N+1).

### Step 3 — Add Items in Order (per module, per week)
For each week, add Canvas items in this exact sequence:
1. Module Intro Quiz (Canvas Quiz, survey mode) — **first item, Module start only**
2. Week N Main Lesson *(Page — deployed via GitHub HTML)*
3. Week N Grammar Lesson 1 *(Page)*
4. Week N Grammar Lesson 2 *(Page)*
5. Week N VR Activity 1 *(Page)*
6. Week N VR Activity 2 *(Page)*
7. Week N Quick Write *(Assignment — completion scored, 5 pts)* OR Discussion *(weeks 3 and 5)*
8. Week N Reading Quiz *(Canvas Quiz — auto-graded, 10 pts)*
9. Module Project *(Canvas Assignment — Tier 3, 50 pts)* — **last item, Module end only**
10. Module Outro Quiz *(Canvas Quiz, survey mode, 5 pts completion)* — **last item**

### Step 4 — Build Canvas Quizzes Manually
For each week's reading quiz:
1. Go to **Quizzes** → **+ Quiz**.
2. Set **Title** (e.g., `Week 1 Reading Quiz — The Odyssey`).
3. Set **Quiz Type**: Graded Quiz | **Points**: 10 | **Time limit**: 15 min.
4. Click **Questions** tab → **+ New Question**.
5. Enter each question from the corresponding `*.quiz.md` file.
6. Mark the correct answer (shown in the quiz file).
7. **Save & Publish**.

For the Module Intro and Outro Quizzes:
1. Go to **Quizzes** → **+ Quiz**.
2. Set **Quiz Type**: Graded Survey (completion credit) or Ungraded Survey.
3. Build questions exactly as specified in the `module-X-intro-quiz.md` file.
4. The quiz **description** field (above the first question) contains the module overview paragraph and the full assignment checklist — paste the provided HTML directly.

### Step 5 — Create Canvas Discussions (weeks 3 and 5 per module)
1. Go to **Discussions** → **+ Discussion**.
2. Paste the prompt from the corresponding week's discussion file.
3. Set **Graded**: Yes | **Points**: 15.
4. Attach the simplified rubric: Substantive post = 15, Thin post = 8, Missing = 0.
5. Set **Reply Required**: one reply to a classmate before full credit.
6. Add to the module at the Week 3/5 position.

### Step 6 — Create Module Projects (Canvas Assignments, Tier 3)
1. Go to **Assignments** → **+ Assignment**.
2. Title: e.g., `Module 1 Project — Epic Hero Essay + ENGAGE Composition`.
3. **Points**: 50.
4. **Submission Type**: Online — Text Entry + File Upload.
5. Create a **Rubric** with these 4 criteria (consistent across all modules):
   - Understanding of the text/material (0–15 pts)
   - Quality of reasoning and argument (0–15 pts)
   - Clarity and precision of expression (0–10 pts)
   - Intellectual honesty — naming limits and opposing views (0–10 pts)
6. Attach the rubric to the assignment.

### Step 7 — Update PDF Links in HTML Files
After uploading PDFs in Step 1, find and replace the placeholder text in every lesson HTML file:
```
[Teacher: insert PDF link here]
```
Replace with the actual Canvas Files download URL.

### Step 8 — Update ENGAGE Session IDs
Find and replace in every VR activity file:
```
[Teacher: insert Session ID]
```
Replace with the actual ENGAGE Session ID from your Teacher Dashboard → Sessions.
